== Description ==

Facebook Messenger Live Chat is a wordpress plugin which will bring facebook messenger to your website.
Visitors of your website can communicate with you directly through facebook messenger without going anywhere.

== Installation ==

After downloading our plugin, please follow the instruction to install
1. From left sidebar click the plugins option> installed plugins
1. Activate “Messenger Live Chat”  plugin by clicking Activate button (Skip step 2 if already activated)
1. Once you have activated, you will see 'Messenger Live Chat' option in the left sidebar
1. Click on “Messenger Live Chat” from left-sidebar
1. Then you will be seen “Authorize” tab for Facebook
1. Please Logged In by clicking Facebook Messenger logo button
1. Once you click, you will be redirected to Facebook login screen
1. After login successfully, you will be redirected to 'Leaddevs Messenger Chat'  app permission to get access of page
1. Once you gave permission to the 'Leaddevs Messenger Chat'  app by clicking the continue button, app will show page list to give permission
1. After giving page permission, you will redirect to our plugin page.
1. Then You have to select 'Settings tab' from the tab
1. Then select one page from drop down list of select page
1. Enable messenger chat of switch button
1. Finally click save button, Your are ready to show messenger chat button of website pages
1. browse any page of your website  like https://messengerchat.leaddevs.com/shop/
1. Then you will see facebook messenger customer  chat button
1. That's it. You/users can chat with that page by Facebook login

= Can I integrate multiple page on my site or on the same page? =

Yes. You can display multiple page by using setting option chosen.

= Are there any security issues with using an Access Token on my site? =

Nope. The Access Token used in the plugin is a “read only” token, which means that it could never be used maliciously to manipulate your Facebook account.
